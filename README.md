## Como rodar a aplicação :arrow_forward:

No terminal, clone o projeto: 

```
https://github.com/Erika-Skarda/integrado

```
<b><p>Backend</p></b>

Para rodar:

```
npm run dev

```
<b><p>MongoDB</p></b>

```
Criar database IntegradoDB

```
